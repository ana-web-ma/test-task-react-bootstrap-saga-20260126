# Тестовое задание: Сайт с постами

## Технологический стек
- React
- React Router
- Redux + Redux Saga
- React Bootstrap
- Axios

## Функциональность

### Страницы приложения:
1. **Main** (`/`) — список всех постов
   - Заголовок, текст, аватар автора для каждого поста
   - Подгрузка комментариев по клику на кнопку
   - Скелетон при загрузке данных (с искусственной задержкой)
   - Скелетон при загрузке комментариев (с искусственной задержкой)
   - Клик по аватару → переход на страницу пользователя

2. **About** (`/about`) — информация о разработчике
   - Сохранение общего layout с хедером

3. **User** (`/user/:userId`)
   - Карточка с информацией о пользователе
   - Список постов только этого пользователя
   - Скелетон при загрузке данных (с искусственной задержкой)
   - Кнопка "Назад" для возврата на главную

4. **Not found** (`/*`)
   - Кастомная страница для несуществующих маршрутов

### Общие элементы:
- **Header** с бургер-меню:
  - При клике открывается боковая панель с навигацией
  - Отображение аватара, имени и email разработчика
  - Ссылки: "Home", "About"
- **Скелетоны** для всех типов контента:
  - Карточка пользователя
  - Посты
  - Комментарии

## Архитектурные решения

### 1. Управление состоянием (Redux + Saga)
- Все асинхронные операции (API запросы) вынесены в саги
- Отдельные редьюсеры для:
  - Постов
  - Комментариев
  - Пользователей

```javascript
<Card>

      <Placeholder xs={6} />
    </Placeholder>
  </Card.Body>

```

### 3. Маршрутизация с обработкой 404
```javascript
<Routes>
  <Route path='/' element={<Layout />}>
    <Route index element={<Mainpage />} />
    <Route path='user/:userId' element={<Userpage />} />
    <Route path='/about' element={<Aboutpage />} />
    <Route path='*' element={<Notfoundpage />} />
  </Route>
</Routes>
```
